# Smart Product Recommendation System

An end-to-end Machine Learning powered Product Recommendation System built using Python and Scikit-learn.

## Features

- Content-Based Recommendation
- Collaborative Filtering
- Hybrid Recommendation
- Product Search
- Streamlit Web App
- FastAPI Backend (Upcoming)

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- FastAPI
- Git & GitHub
